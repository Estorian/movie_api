Return all user data.
Endpointurl: '/users'
Method: GET
Request data: none
Response data: JSON

Return data about a specific user.
Endpoint url: '/users/[username]'
Method: GET
Request data: none
Response data: JSON

Allow new users to register
Endpoint url: '/register'
Method:POST
Request data: JSON object with the following keys:
{
username: String,
password: String,
email: String,
Birthday: Date
}
Response data: plain text response: "User [username] successfully registered."

Allow users to update their user info.
Endpoint url: '/users/[username]'
Method: PUT
Request data: JSON object with the following keys with the data to be updated:
{
username: String,
password: String,
email: String,
Birthday: Date
}
Response data: JSON of new user information

Allows users to add a movie to their list of favorites.
Endpoint: '/users/[username]/movies/[movieID]'
Method: PUT
Request Data: none
Response data: JSON of updated user data

Allows users to remove a movie from their list of favorites.
Endpoint: '/users/[username]/movies/[movieID]/remove'
Method: DELETE
Request Data: none
Response data: JSON of updated user data

Allows users to deregister.
Endpoint: 'users/[username]'
Method: DELETE
Request Data: none
Response data: plaintext response "[username] was deleted."
